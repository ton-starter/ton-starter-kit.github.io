# TON Starter Kit

Ваш путеводитель в экосистеме The Open Network.

---

## Быстрый старт

1. **Установите зависимости**
   ```bash
   npm install
   ```
2. **Запустите проект**
   ```bash
   npm run dev
   ```
3. **Откройте в браузере**
   ```text
    http://localhost:3030
   ```

---

## О проекте

Простой и понятный портал-навигатор по экосистеме TON. Главная цель — дать новичку ответы на ключевые вопросы за 30 секунд и направить в нужную часть экосистемы.

## Технологии

    Nuxt 3

    Element Plus

    TypeScript
